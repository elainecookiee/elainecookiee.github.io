ABOUT

Elaine Learns to Code Project. Similar to Jennifer Dewalt's project, I will be publishing all my code as an open source on GitHub. Instead of writing one blog post a day to accompany each site, I will be writing a detailed blog post each week about my thought and implementation process from brainstorming ideas, sketching, wire framing to the resources that I find are actually useful.

Some ideas for projects I am hoping to work on are redesigning apps, UI designs, logos, posters, business cards, packaging designs, e-commerce site, resume site.

Background:

  • UW Grad, Honours Psychology Spring 2015.
  • Self taught HTML, CSS, a bit of Python and Javascript (beginner level).
  • Began with no experience using web design tools or programming.

What I want to get out of this:

  • The best practice I could ever get.
  • A hand full of tools that would help me plan, design and make websites.
  • Create interactive sites and become familiar with Javascript, JQuery and other programming languages.
  • Explore design away from traditional illustration.
  • Create something visually pleasing, easy and delightful to use.
